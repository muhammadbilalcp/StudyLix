Integration Guide

How to connect this portal to your main app.

Step 1 — Add this function to index.html

Paste this inside your <script> tag in index.html:

async function loadTeacherContent() {
  try {
    // Load teacher-added classes (overrides hardcoded CLASSES)
    const cSnap = await db.collection('admin_classes')
                          .orderBy('createdAt','desc').get();
    if (!cSnap.empty) {
      CLASSES.length = 0;
      cSnap.docs.forEach(d => {
        CLASSES.push({ id: d.id, ...d.data() });
      });
    }

    // Load teacher-added questions (merges into QB)
    const qSnap = await db.collection('admin_questions').get();
    qSnap.docs.forEach(d => {
      const q = d.data();
      if (!QB[q.subject]) QB[q.subject] = [];
      if (!QB[q.subject].find(x => x.q === q.q))
        QB[q.subject].push({ q: q.q, opts: q.opts, ans: q.ans });
    });
  } catch(e) { console.warn('Teacher content:', e); }
}

Step 2 — Call it after loadUserData()

Find this line in your auth.onAuthStateChanged block:

await loadUserData();

Add this line directly after it:

await loadTeacherContent();

That's it! ✅

Classes and questions you add here will automatically appear in the student app. No other changes needed. The built-in hardcoded content remains as a fallback if Firestore is empty.
